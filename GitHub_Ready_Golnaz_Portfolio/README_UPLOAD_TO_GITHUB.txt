GITHUB PAGES UPLOAD INSTRUCTIONS

Upload ALL these items to your GitHub repository root:

1. index.html
2. assets/ folder
3. pdfs/ folder

Do not upload only index.html.
Do not rename the folders unless you also edit the links inside index.html.
The correct repository structure is:

portfolio/
  index.html
  assets/
    historic_01.jpg
    historic_02.jpg
    ...
    revit_07.jpg
  pdfs/
    Historic_Facade_Survey_Web_Portfolio.pdf
    Revit_BIM_University_Project_Web_Portfolio.pdf

After uploading, enable GitHub Pages:
Settings > Pages > Source: Deploy from a branch > Branch: main > Folder: /root > Save

Your final public link will look like:
https://YOUR-USERNAME.github.io/portfolio/

Test the final public link in an incognito/private browser before putting it in your CV.
